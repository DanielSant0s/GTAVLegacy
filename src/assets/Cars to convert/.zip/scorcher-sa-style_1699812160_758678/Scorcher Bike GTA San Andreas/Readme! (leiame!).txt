|------Bicycle Scorcher GTA SA------|

[----------Português PT-BR------------]

- Caso deseja instalar de forma normal (substituindo), entre na pasta "modloader", escolha uma pasta com a bike
a seu gosto e jogue no modloader de seu GTA, caso tenha o CLEO, mova o arquivo "scorcher_name" para o cleo_text.

- Caso deseje instalar sem substituição, vá para a pasta "add. without replacing" onde se encontra os dois modelos do mod,
as manipulações handling e com um "[ID]" para adicionar um número a sua escolha e o .fxt para aparecer o nome da bicicleta no jogo, 
recomendo que adicione para melhor experiência.

- Não precisa adulterar nada no "cargrp.dat", os nomes já estão lá para aparecerem com NPCs.

- Por favor, caso reposte o Mod ou poste em outro site, deixe os créditos do autor!!

[-----------English USA-------------]

- If you want to install normally (replacing), go to the "modloader" folder, choose a folder with the bike
to your liking and play in your GTA modloader, if you have CLEO, move the "scorcher_name" file to cleo_text.

- If you want to install without replacement, go to the indicated folder where the two models of the mod are located,
the handling manipulations and with an "[ID]" to add a number of your choice and the .fxt to display the name of the bike in the game,
I recommend you add it for a better experience.

- You don't need to change anything in "cargrp.dat", the names are already there to appear with NPCs.
